The setup problem every shop knows.

Traditional setups can force machinists to stop, move the part, re-indicate, shim, block, and run the same job in multiple positions. Every extra move adds time, risk, and opportunity for error.

The Libert Elevation System is built to change that.

By raising and supporting the workpiece with modular components, the system creates clearance underneath and around the part so more machining can happen with fewer interruptions.

Built to hold work higher, cleaner, and with more access.
